 

Quiz App

Build a quiz app with more than one question which has multiple options with only one correct answer. One mark for each right answer.
Admin should be able to add quizzes and questions from the backend.
Users should be able to select a quiz which they wish to attend from frontend and should be able to see right and wrong answers and the total score after completion.

Dashboard functionalities

1. Primary
a. Add quiz and questions
b. Store user scores

2. Secondary
a. Export user scores as csv



Frontend functionalities

1. Primary
a. Basic user authentication (Login/Logout)
b. Quiz listing page
c. Quiz form
d. Submitting quiz form will redirect to a success page where users will be able to see the right and wrong answers he did and the total marks scored.

2. Secondary
a. User authentication with email not username.


RestAPI

1. Api to get user data, which includes the quizzes they have attended and the marks scored.


NB: 
1. All the primary tasks are mandatory and secondary tasks are optional for candidates who have less than 5 years of experience.
2. Both primary and secondary tasks are mandatory for candidates experience of 5 or more years.
3. Task duration: 2h 30 m
4. Styling is not mandatory.
5. Candidates can complete maximum functionalities within the time limit.
6. Candidates who complete more tasks (including primary and secondary tasks) within the time limit will get weightage.
